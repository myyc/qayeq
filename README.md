<div align="center">A minimal WebKit-based browser for GNOME.</div>

<br />

Everything vanishes. F1 and Esc to toggle the sidebar
or the top bar. Usual shortcuts otherwise for tabs and
navigation.

To build it you need the Rust toolchain and WebKit.
Or just download the flatpak.

It's nowhere close to replacing any browser right now.
It might never get to that stage.
